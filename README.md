## 📁 Core Featured Projects

### 🏥 Clinical Decision Support System (ASR + RAG)
An advanced decision-support ecosystem tailored for intensive medical environments, combining localized speech recognition with knowledge-aware verification layers.
- **Backend Core:** Developed processing pipelines using a recursive RAG structure, a context builder, a verifier, and a dedicated query router to prevent hallucinations.
- **Medical Intelligence:** Integrated dynamic medical drug-matching components (`drug_lexicon.py`) alongside local medical datasets, including Vietnamese ICD-10 codebooks, LOINC parameters, and anaphylaxis protocols (`tt51_phan_ve.md`).
- **Data Interoperability:** Architected a standardized data integration client utilizing the international FHIR protocol (`fhir_client.py`) to smoothly handle patient resource records.
- **Frontend:** Built a React ecosystem featuring a comprehensive patient manager dashboard and synchronous speech chat workspaces.

### 🎙️ Speech-to-Text Extension for Online Classes
A production-ready live captioning and educational transcript indexing platform deployed as a browser extension.
- **Audio Capture Infrastructure:** Constructed low-latency browser audio streaming capture utilities linked to a custom backend transcription server.
- **VAD Processor:** Implemented an optimized Voice Activity Detection (`VAD`) processor to separate background noise and handle silence truncation efficiently before feeding into speech-to-text engines.
- **Evaluation Sandbox:** Programmed a dual-model evaluation framework (`evaluate_dual_model.py`) to systematically measure transcription accuracy against a set of baseline ground truths.

### 🤖 "Guess the MV" Discord Bot Game
An event-driven automated utility bot built to run fully functional music gamification channels inside server rooms.
- Built multi-threaded crawling script architectures to gather video/track indices and map text similarities.
- Created custom masking and fuzzy string-matching engines (`masking.py`, `matching.py`) to gauge user performance thresholds.
